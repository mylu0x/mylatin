# Serbian
Serbian already uses Gaj’s Latin alphabet, so Rule B isn’t really needed...

## Rule codes
- Rule A: `mylatin-sr-a` or `mylatin_sr_a`
- Rule B: `mylatin-sr-b` or `mylatin_sr_b`
